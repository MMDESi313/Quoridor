## ✨ Overview

This project is a modern web implementation of the classic **Quoridor** board game.

Rather than focusing only on the user interface, this project focuses on implementing a complete and reusable **game engine** that contains all of the game's core mechanics.

The current version supports **local multiplayer** on a single device.

---

## ✨ Features

- 🎮 Complete implementation of Quoridor rules
- 👥 Local multiplayer (2 players)
- 🚶 Legal move validation
- 🧱 Wall placement validation
- 🛣️ Pathfinding verification
- 🔄 Turn management
- 🏆 Winner detection
- 📱 Responsive interface

```
        │
 ┌──────┼───────────────┐
 │      │               │
Moves Walls Pathfinding
 │      │               │
 └──────┼───────────────┘
        │
 Winner Detection
```

The game logic is completely separated from the UI, making the engine easier to maintain, extend, and test.

---

## 🧠 Game Engine

The game engine was designed and implemented from scratch.

It is responsible for:

- Validating player movement
- Validating wall placement
- Ensuring both players always have a valid path
- Managing turns
- Detecting the winner
- Updating the game state

AI was occasionally used as a learning and brainstorming assistant during development, but the game engine implementation itself was written manually.
