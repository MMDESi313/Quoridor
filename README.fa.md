# 🎮 بازی Quoridor

**یک پروژه شخصی برای یادگیری.**

پیاده‌سازی تحت وب بازی کلاسیک **Quoridor** با استفاده از **React** و **TypeScript**.

## ✨ معرفی

این پروژه یک پیاده‌سازی تحت وب از بازی رومیزی **Quoridor** است.

تمرکز اصلی این پروژه صرفاً ساخت رابط کاربری نبوده، بلکه طراحی و پیاده‌سازی یک **موتور بازی (Game Engine)** کامل و مستقل بوده که تمامی قوانین اصلی بازی را مدیریت می‌کند.

نسخه فعلی از **بازی دو نفره روی یک دستگاه** پشتیبانی می‌کند.

```text

        │
        ▼

        │

      موتور بازی
        │
 ┌──────┼───────────────┐
 │      │               │
حرکت‌ها دیوارها بررسی مسیر
 │      │               │
 └──────┼───────────────┘
        │
    تشخیص برنده
```

تمام منطق بازی از رابط کاربری جدا شده تا توسعه، نگهداری و تست پروژه ساده‌تر باشد.

## 🧠 موتور بازی

موتور بازی از ابتدا توسط خودم طراحی و پیاده‌سازی شده است.

وظایف این بخش شامل موارد زیر است:

- اعتبارسنجی حرکت بازیکنان
- اعتبارسنجی قرار دادن دیوار
- اطمینان از وجود حداقل یک مسیر معتبر برای هر دو بازیکن
- مدیریت نوبت‌ها
- تشخیص برنده
- به‌روزرسانی وضعیت بازی

در طول توسعه، در برخی بخش‌ها برای یادگیری یا بررسی ایده‌ها از **AI** کمک گرفتم، اما پیاده‌سازی موتور بازی و منطق آن به‌صورت دستی انجام شده است.
